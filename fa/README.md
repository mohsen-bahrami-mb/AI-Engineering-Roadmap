# چرا این مخزن ایجاد شد؟

بیشتر Roadmapهای هوش مصنوعی تمرکز زیادی روی مباحث تئوری، ریاضی و پژوهشی دارند.

اما واقعیت بازار کار امروز چیز دیگری است.

اکثر شرکت‌ها مدل‌هایی مثل GPT را از صفر آموزش نمی‌دهند.

بلکه روی ساخت موارد زیر تمرکز دارند:

- AI Assistant
- RAG Systems
- Enterprise Knowledge Systems
- AI Agents
- Workflow Automation
- Document Intelligence
- Multimodal Applications

هدف این مخزن آموزش مهارت‌هایی است که واقعاً در آگهی‌های شغلی AI Engineer موردنیاز هستند.

# RAG (بازیابی و تولید)

## چرا RAG؟

RAG به مدل اجازه می‌دهد به دانشی دسترسی داشته باشد که در زمان آموزش وجود نداشته است.

مثال‌ها:

- مستندات شرکت
- PDFها
- ویکی داخلی
- مستندات محصول
- سیستم پشتیبانی مشتری

---

## RAG در برابر Fine-Tuning

| RAG | Fine-Tuning |
|---------|---------|
| دانش جدید فوری | نیازمند آموزش مجدد |
| ارزان‌تر | گران‌تر |
| مناسب دانش | مناسب تغییر رفتار |
